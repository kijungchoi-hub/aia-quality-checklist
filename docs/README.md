# 문서 색인

## 폴더 안내

- `checklists/`: 최종 체크리스트와 표 형식 산출물
- `projects/`: 프로젝트별 작업 문서와 추적 자료
- `reports/`: LDS 보고서, 슬라이드 초안, 분석 메모
- `governance/`: 운영 규칙, 릴리스 게이트, 장애 분류 문서
- `templates/`: 재사용 가능한 운영 템플릿
- `archive/`: 이전 버전과 백업 자료

## 권장 규칙

- 확정된 산출물은 `checklists/`에 둡니다.
- 진행 중인 배포 자료는 `projects/`에 둡니다.
- 일회성 보고서와 발표 초안 자료는 `reports/`에 둡니다.
- 프로세스 및 정책 문서는 `governance/`에 둡니다.
