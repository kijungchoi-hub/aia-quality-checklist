# 검증 템플릿 안내


---

## 목적

이 폴더는 변경 전후 품질 차이와 회귀 여부를 검증하기 위한 템플릿을 모아 둔 영역입니다.

---

## 포함 파일

| 파일 | 구분 | 설명 |
|---|---|---|
| [`22_evaluation_dataset.md`](22_evaluation_dataset.md) | Template | 평가셋/검증셋 관리 문서 |
| [`22_evaluation_dataset_sample.csv`](22_evaluation_dataset_sample.csv) | Sample | 평가셋/검증셋 입력 예시 |
| [`23_manual_auto_review_log.md`](23_manual_auto_review_log.md) | Template | 수동+자동 평가 통합 기록 문서 |
| [`23_manual_auto_review_log_sample.csv`](23_manual_auto_review_log_sample.csv) | Sample | 수동+자동 평가 입력 예시 |
| [`24_expansion_validation_matrix.md`](24_expansion_validation_matrix.md) | Template | 타 영역 확장 검증 매트릭스 |
| [`24_expansion_validation_matrix_sample.csv`](24_expansion_validation_matrix_sample.csv) | Sample | 확장 검증 매트릭스 입력 예시 |
| [`25_regression_test_tracker.md`](25_regression_test_tracker.md) | Template | 회귀 테스트 관리 문서 |
| [`25_regression_test_tracker.csv`](25_regression_test_tracker.csv) | Input | 회귀 테스트 입력 시트 |
| [`26_regression_test_tracker_sample.md`](26_regression_test_tracker_sample.md) | Sample | 회귀 테스트 작성 예시 |
| [`26_regression_test_tracker_sample.csv`](26_regression_test_tracker_sample.csv) | Sample | 회귀 테스트 입력 예시 |

---

## 사용 시점

1. 검색 또는 생성 개선 후 전후 비교가 필요할 때
2. 배포 전 회귀 여부를 확인할 때
3. 품질 개선 효과를 수치와 사례로 함께 검증할 때

---

## 사용 방식

- `*.md`는 설명형 템플릿입니다.
- `*.csv`는 입력형 템플릿입니다.
- `*_sample.md`, `*_sample.csv`는 작성 예시입니다.
