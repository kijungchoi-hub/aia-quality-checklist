# 운영 실행 템플릿 안내


---

## 목적

이 폴더는 품질 개선 실험을 설계하고, 검색 실패와 생성 실패를 품질 운영 관점에서 축적하기 위한 템플릿을 모아 둔 영역입니다.

---

| 파일 | 구분 | 설명 |
|---|---|---|
| [`16_experiment_design.md`](16_experiment_design.md) | Template | 실험 설계 문서 |
| [`16_experiment_design.csv`](16_experiment_design.csv) | Input | 실험 설계 입력 시트 |
| [`17_experiment_design_sample.md`](17_experiment_design_sample.md) | Sample | 실험 설계 작성 예시 |
| [`17_experiment_design_sample.csv`](17_experiment_design_sample.csv) | Sample | 실험 설계 입력 예시 |

| [`19_search_failure_codebook_sample.md`](19_search_failure_codebook_sample.md) | Sample | 검색 실패 코드북 예시 |

---

## 사용 시점

1. 실험을 시작할 때
2. 검색 실패 원인을 분류할 때
3. 검색 성공 후 생성 실패를 별도 기록할 때

---

## 사용 방식

- `*.md`는 설명형 템플릿입니다.
- `*.csv`는 입력형 템플릿입니다.
- `*_sample.md`, `*_sample.csv`는 작성 예시입니다.
