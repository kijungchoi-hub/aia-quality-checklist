# 배포 템플릿 안내


---

## 목적

이 폴더는 운영 반영 승인과 배포 판단을 표준화하기 위한 템플릿을 모아 둔 영역입니다.

---

## 포함 파일

| 파일 | 구분 | 설명 |
|---|---|---|
| [`27_release_gate_checklist.md`](27_release_gate_checklist.md) | Template | 배포 게이트 문서 |
| [`27_release_gate_checklist.csv`](27_release_gate_checklist.csv) | Input | 배포 점검 입력 시트 |
| [`28_release_gate_checklist_sample.md`](28_release_gate_checklist_sample.md) | Sample | 배포 게이트 작성 예시 |
| [`28_release_gate_checklist_sample.csv`](28_release_gate_checklist_sample.csv) | Sample | 배포 점검 입력 예시 |

---

## 사용 시점

1. 배포 승인 여부를 판단할 때
2. 보류 사유를 명확히 남겨야 할 때
3. 운영 반영 기준을 문서화해야 할 때

---

## 사용 방식

- `*.md`는 설명형 템플릿입니다.
- `*.csv`는 입력형 템플릿입니다.
- `*_sample.md`, `*_sample.csv`는 작성 예시입니다.
