# Answer Quality Governance Assets

이 폴더는 `answer` 품질 개선 프로젝트의 운영 기준서와 관리 문서를 보관한다.

## 포함 문서

| 파일명 | 용도 |
|---|---|
| `answer_quality_operating_framework.md` | 전체 운영 프레임워크 |
| `answer_search_failure_codebook.md` | 검색 실패 분류 기준 및 초안 사례 |
| `answer_generation_failure_casebook.md` | 생성 실패 사례 초안 |
| `answer_manual_auto_review_log.md` | 수동/자동 평가 통합 로그 초안 |
| `answer_release_gate_checklist.md` | 운영 반영 전 점검표 |
| `answer_weekly_quality_report_2026-W14.md` | 현재 프로젝트 기준 주간 보고 초안 |
| `answer_expansion_validation_matrix.md` | 타 영역 확장 검증 초안 |

## 운영 원칙

- 평가 기준, 실패 분류, 회귀 통제 문서는 항상 함께 갱신한다.
- 문서명은 유지하고 버전 또는 주차는 본문 메타 정보로 관리한다.
- 배포 전에는 최소 `평가셋`, `실패 로그`, `회귀셋`, `배포게이트` 4종 확인을 권장한다.
