# Reporting 템플릿 안내


---

## 목적

이 폴더는 주간 품질 운영 결과를 정리하고 공유하기 위한 템플릿을 모아둔 영역이다.

---

## 포함 파일

| 파일 | 구분 | 설명 |
|---|---|---|
| `weekly_quality_report.md` | Template | 주간 보고 문서 |
| `weekly_quality_report.csv` | Input | 주간 보고 입력 시트 |
| `weekly_quality_report_sample.md` | Sample | 주간 보고 작성 예시 |
| `weekly_quality_report_sample.csv` | Sample | 주간 보고 입력 예시 |

---

## 사용 시점

1. 주간 품질 운영 결과를 보고할 때
2. 실패 사례와 차주 계획을 함께 정리할 때
3. 운영 회고 기록을 표준 포맷으로 남길 때

---

## 사용 방식

- `*.md`는 설명형 템플릿이다.
- `*.csv`는 입력형 템플릿이다.
- `*_sample.md`, `*_sample.csv`는 작성 예시다.
