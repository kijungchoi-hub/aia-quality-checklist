# 생성 실패 케이스북 템플릿 (Generation Failure Casebook Template)


---

## 목적

검색 성공 이후에도 발생하는 생성 실패를 별도 관리한다.

---

## 실패 유형 정의

| 코드 (Code) | 유형 (Type) | 설명 (Description) |
|---|---|---|
| HALL | Hallucination | 근거 없는 정보 생성 |
| OMIT | Omission | 핵심 조건 또는 답변 누락 |
| DIST | Distortion | 근거는 있으나 의미 왜곡 |
| FORM | Format | 응답 형식 지시 불이행 |
| CITE | Citation | 근거/출처 연결 오류 |

---

---

## 예시 (Example)

| 샘플 ID | Chuck Hit | 생성 결과 | 실패 코드 | 원인 메모 | 개선 액션 | 재검 결과 |
|---|---|---|---|---|---|---|
| GN-001 | Y | 조건 3개 중 1개 누락 | OMIT | 근거는 있으나 요약 과정에서 일부 탈락 | 조건 나열 프롬프트 추가 | 개선 |
| GN-002 | Y | 근거 문서와 다른 예외 조건 설명 | DIST | 상충 문장 혼합 | 출처 우선 문장 제약 추가 | 재검 필요 |
| GN-003 | Y | 근거 없는 일반 규정 생성 | HALL | fallback 규칙 부재 | 근거 없으면 답변 보류 규칙 추가 | 개선 |

- 가장 빈번한 생성 실패: `OMIT`
- 반복되는 응답 패턴: 조건형 답변에서 핵심 조항 누락
- 프롬프트 수정 필요 여부: 필요
