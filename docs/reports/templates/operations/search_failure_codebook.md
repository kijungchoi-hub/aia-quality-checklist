# 검색 실패 코드북 템플릿 (Search Failure Codebook Template)


---

## 목적

검색 실패를 `데이터`, `색인`, `메타데이터`, `질의`, `랭킹` 관점으로 분리 기록한다.

---

## 코드 정의

| 코드 (Code) | 유형 (Type) | 설명 (Description) | 예시 (Example) |
|---|---|---|---|
| DATA | 데이터 | 정답 문서 또는 정답 매핑 오류 | 정답 문서 자체 누락 |
| INDEX | 색인 | 문서/Chuck 색인 누락 또는 갱신 실패 | 최신 문서 미색인 |
| META | 메타데이터 | 필터 또는 속성값 오류 | 잘못된 카테고리 매핑 |
| QUERY | 질의 | 질의 해석 실패 또는 확장 부족 | 약어 질의 미해석 |
| RANK | 랭킹 | 정답 문서는 있으나 순위가 낮음 | Top-10 밖 위치 |
| NOISE | 노이즈 | 무관 문서가 과다 유입됨 | 상위 결과 대부분 무관 |

## 예시 (Example)

| 샘플 ID | 질문 (Query) | 기대 문서 | 실제 상위 결과 | 실패 코드 | 원인 메모 | 후속 액션 |
|---|---|---|---|---|---|---|
| SR-001 | 해지 수수료는 언제 발생하나요 | 서비스 이용약관 12조 | FAQ 일반 안내, 공지사항 | RANK | 정답 문서는 존재하나 상위 10위 밖 | BM25 가중치 상향 |
| SR-002 | 미성년자 가입 제한이 있나요 | 가입정책 3조 | 관련 문서 없음 | INDEX | 최신 가입정책 문서 색인 누락 | 색인 재생성 |
| SR-003 | VIP 등급 기준 알려줘 | 멤버십 정책 2장 | 카드혜택 소개 문서 다수 | META | 카테고리 메타값 오류 | 메타데이터 수정 |

- 가장 많은 실패 코드: `RANK`
- 반복 질의군: 약관 조건 비교, 정책 예외 조항
- 우선 개선 대상: BM25 조합 및 메타데이터 정비
