# Validation 템플릿 안내


---

## 목적

이 폴더는 변경 전후 품질 차이와 회귀 여부를 검증하기 위한 템플릿을 모아둔 영역이다.

---

## 포함 파일

| 파일 | 구분 | 설명 |
|---|---|---|
| `regression_test_tracker.md` | Template | 회귀 테스트 관리 문서 |
| `regression_test_tracker.csv` | Input | 회귀 테스트 입력 시트 |
| `regression_test_tracker_sample.md` | Sample | 회귀 테스트 작성 예시 |
| `regression_test_tracker_sample.csv` | Sample | 회귀 테스트 입력 예시 |

---

## 사용 시점

1. 검색 또는 생성 개선 후 전후 비교가 필요할 때
2. 배포 전 회귀 여부를 확인할 때
3. 품질 개선 효과를 수치와 사례로 함께 검증할 때

---

## 사용 방식

- `*.md`는 설명형 템플릿이다.
- `*.csv`는 입력형 템플릿이다.
- `*_sample.md`, `*_sample.csv`는 작성 예시다.
