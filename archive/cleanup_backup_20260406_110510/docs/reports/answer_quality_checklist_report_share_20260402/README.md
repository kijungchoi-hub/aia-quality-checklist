## 목적 (Purpose)

이 폴더는 `answer` 품질 체크리스트 기반 공유 자료를 정리한 사본 패키지입니다.
최종 보고서 본문과 함께 요약본, 기초자료, 주간 보고, 미착수 개선 사례, 발표용 참고 문구를 별도 파일로 정리했습니다.

---

## 파일 안내 (File Guide)

- `01_answer_quality_checklist_report_final.md`: 공유용 최종 보고서 본문
- `01a_answer_quality_checklist_report_final_official.md`: 더 공식적인 보고서 문체로 정리한 버전
- `01b_answer_quality_checklist_report_summary.md`: 요약 보고 버전
- `02_answer_quality_checklist_onepage_summary.md`: 1페이지 요약본
- `03_answer_quality_checklist_report_base.md`: 보고서 기초자료와 수치 정리
- `04_answer_quality_checklist_weekly_report.md`: 주간 운영 관점의 보고 예시
- `05_answer_quality_checklist_unstarted_improvement_cases.md`: 미착수 항목 중심의 실제 개선 사례 예시
- `06_answer_quality_checklist_final_copy_reference.md`: 발표 자료 문구 참고본
- `07_answer_quality_framework_playbook.md`: 품질 개선 Framework의 실제 적용 절차와 운영 기준을 정리한 기법서

---

## 사용 메모 (Usage Notes)

- 본 폴더는 공유용 사본이므로 원본 수정은 `docs/reports/`의 원문 파일에서 진행하는 것을 권장합니다.
- 발표보다 문서 공유를 우선할 경우 `01a`와 `01b`를 먼저 전달하면 공식 보고와 요약 보고를 함께 제공할 수 있습니다.
